# Google Link

## Описание
Tampermonkey userscript для Waze Map Editor (WME), который автоматически ищет и привязывает Google POI по адресу выбранного POI.

## Структура
- `src/wme-google-link.user.js` — основной скрипт

## Как работает
1. При клике на POI в WME скрипт получает адрес через WME SDK
2. Строит поисковый запрос из улицы + номера дома + города + страны
3. Находит секцию "Linked Google POIs" в панели редактирования
4. Добавляет кнопку "🔍 Google Link" которая:
   - Автозаполняет поле поиска адресом POI (если нашёл input)
   - Или открывает Google Maps с поиском (fallback)

## Технологии
- WME JavaScript SDK (`getWmeSdk`)
- MutationObserver для отслеживания DOM
- Событие `wme-selection-changed` и `wme-feature-editor-opened`
