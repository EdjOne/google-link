# 🔍 Google Link (WME)

Tampermonkey скрипт для [Waze Map Editor](https://www.waze.com/editor), який автоматично шукає Google POI за адресою venue.

🔗 [GitHub](https://github.com/EdjOne/google-link) | 📦 [Встановити](https://raw.githubusercontent.com/EdjOne/google-link/master/src/wme-google-link.user.js)

## Що робить

1. Клікни на POI в WME
2. Скрипт бере адресу (вулиця + номер + місто)
3. Шукає в Google Places
4. Показує результати з відстанню
5. **🔗 Link** — прив'язує Google Place до venue

## Встановлення

1. Встанови [Tampermonkey](https://www.tampermonkey.net/)
2. Перейди на сторінку скрипта → натисни **Raw** → підтверди встановлення
3. Відкрий WME — з'явиться вкладка **🔍 GL**

## Як користуватися

1. **Клікни на POI** на карті
2. З'явиться панель **"🔍 Google Link"**
3. Скрипт автоматично шукає Google за адресою
4. **Результати** з кольоровою індикацією відстані:
   - 🟢 < 50 м — точно те саме
   - 🟡 < 300 м — поруч
   - 🔴 > 300 м — далеко
5. **Клікни на результат** → автоматично заповнює поле пошуку в Google
6. Перевір і **збережи** (Ctrl+S)

## Налаштування (вкладка 🔍 GL)

- **📍 Відстань** — показувати відстань до кожного результату
- **🔗 Тільки без посилань** — підсвічує cyan POI без Google прив'язки
- **Радіус** — максимальна відстань пошуку (за замовчуванням 5000 м)

## Фільтри

Скрипт автоматично **пропускає**:
- 🌿 Природу (NATURAL_FEATURES) — острови, озёра, ліси, річки, болота
- 🅿️ Парковки (PARKING_LOT)

Ці POI не підсвічуються та не відкривають панель при кліку.

## Обмеження

- Працює тільки з **POI** (venue), не з адресами/шляхами
- POI **має мати адресу** (вулиця + номер будинку)
- Google API без ключа (~100 запитів/день)
- Після прив'язки — **збережи** зміни вручну
