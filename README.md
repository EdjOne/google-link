# 🔍 Google Link (WME) — Інструкція

## Що робить

Автоматично шукає POI в Google Maps за адресою обраного WME-поінта та прив'язує Google Place ID одним кліком.

## Встановлення

1. Встанови [Tampermonkey](https://www.tampermonkey.net/) у браузер
2. Відкрий сторінку скрипта на GitHub → натисни **"Raw"** → Tampermonkey запропонує встановити
3. Відкрий [WME](https://www.waze.com/editor) — з'явиться вкладка **🔍 GL** в лівій панелі

## Як користуватися

1. **Обери POI** на карті (клік на іконку)
2. З'явиться плаваюча панель **"🔍 Google Link"** справа
3. Скрипт автоматично шукає в Google за адресою: *вулиця, №, місто, країна*
4. **Результати** — список Google-місць з відстанню:
   - 🟢 < 50 м — точно те саме
   - 🟡 < 300 м — поруч
   - 🔴 > 300 м — далеко
5. **Клікни на результат** — скрипт автоматично:
   - Відкриє секцію "Зовнішні сервіси" в редакторі
   - Натисне кнопку "Прив'язати до Google"
   - Введе адресу в пошук Google
   - Обере перший варіант
6. **Перевір** результат і **збережи** (Ctrl+S)

## Налаштування (вкладка 🔍 GL)

| Налаштування | Що робить |
|---|---|
| **📍 Відстань** | Показувати відстань до кожного результату (кольорова індикація) |
| **🔗 Тільки без посилань** | Підсвічує cyan на карті POI, які ще не прив'язані до Google. При пошуку — реагує тільки на такі POI |
| **Радіус: ___ м** | Максимальна відстань пошуку (за замовчуванням 5000 м) |

## Підсвітка нелinkованих POI

Якщо ввімкнено **"🔗 Тільки без посилань"**:
- 🔵 **Іконки** на карті отримують cyan-обводку
- 🔵 **Текстові мітки** стають cyan + жирний + свічення
- Подсвітка **оновлюється автоматично** при:
  - Зумі / переміщенні карти
  - Збереженні POI

## Обмеження

- Працює тільки з **POI** (venue), не з адресами/шляхами
- POI **має мати адресу** (вулиця + номер будинку)
- Google API використовується без ключа (обмеження: ~100 запитів/день)
- Після прив'язки — **збережи** зміни вручну (Ctrl+S)

## Вимкнення

У Tampermonkey → Google Link → вимкни скрипт, або видали `gl_enabled` з localStorage.
